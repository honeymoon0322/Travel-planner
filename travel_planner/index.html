네이버 지도 클라이언트 ID가 필요해요

네이버 지도 API는 네이버 클라우드 플랫폼에서 발급받은 키와, 사용할 도메인 등록이 필요합니다. 무료 한도(월 1,000만 호출)로 충분히 사용할 수 있어요.

  1. 네이버 클라우드 플랫폼 가입 · 로그인
  2. Console → Services → Maps → Application 등록
  3. Web Dynamic Map 체크, 사용할 도메인 추가
  4. 발급된 Client ID(= Key ID)를 아래에 입력
키는 이 페이지에만 사용되고 저장되지 않아요.
